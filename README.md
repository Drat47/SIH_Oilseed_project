# SIH Oilseed Project

This repository contains the complete Smart India Hackathon (SIH) Oilseed Project,
including both backend and frontend components.

## Project Structure

SIH_Oilseed_project
├── backend
│ ├── FastAPI backend
│ ├── ML models and prediction logic
│ └── requirements.txt
│
├── shadcn-ui
│ ├── React + shadcn UI frontend
│ └── User interface for dashboards and advisories
│
└── .gitignore

markdown
Copy code

## Backend
- Built using FastAPI
- Includes ML-based advisory and yield prediction
- Virtual environments are excluded from version control

## Frontend
- Built using React with shadcn UI
- Communicates with backend APIs
- Node dependencies are excluded from Git

## Setup (Local)
1. Clone the repository
2. Setup backend using `requirements.txt`
3. Setup frontend using `npm install`
4. Run backend and frontend separately

This repository is maintained as a single clean project for SIH submission.
